Samaritan

Samaritan is intended to be a Intelligent Video Surveillance System which has various techniques inherited from Computer Vision and Machine Learning.

View project on GitHub

Objective

In this project we intend to design and implement an Intelligent Video Surveillance system which consists of an embedded system and PC-based system. We will be designing a PC-based intelligent system that detects motion and depending on a threshold value for movement and activates alerts(Sound/Email) in presence of motion. Samaritan is implemented on an embedded system consisting of an ARM processor(Raspberry Pi) which has sufficient amount of RAM(~512MB).

Hardware

  • Raspberry Pi Model B
  • 8GB SD Card
  • USB WiFi Adapter for Raspberry Pi
  • USB Camera/Pi camera module
  • PC or Laptop
  • Stepper Motor (Optional)
  • Motor Driver (Optional)
  • Router (Optional)
